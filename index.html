A modern
Desktop Client
for Kong Admins

Primate (previously known as KongDash)
is a stand-alone application that helps
manage Kong API Gateway with its Admin API.

Easy. Straightforward.

No Sophisticated setup procedures to start
managing your Kong instances. All you need to do is:

system_update_alt

Download the installer
for your platform.

settings_suggest

Run the installer to setup
Primate in your computer.

dvr

Run Primate just like any
other desktop application.

Manage Multiple Instanes
of Kong Gateway Servers

Primate stores the connection information in your computer everytime a new connection is added.
